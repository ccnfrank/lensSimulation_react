PT
Webapp desenvolvido em React que simula o comportamento de diferentes lentes de câmera (distância focal, abertura e profundidade de campo). O objetivo é ajudar fotógrafos iniciantes a entenderem como a escolha do equipamento impacta o resultado final da imagem.

EN
Web app developed in React that simulates the behavior of various camera lenses (focal length, aperture, and depth of field). The goal is to help beginner photographers understand how equipment choices impact the final image.


DE
In React entwickelte Web-App, die das Verhalten verschiedener Kameraobjektive simuliert (Brennweite, Blende und Schärfentiefe). Das Ziel ist es, Anfängern in der Fotografie zu helfen, zu verstehen, wie die Wahl der Ausrüstung das endgültige Bild beeinflusst.
